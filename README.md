# Retail Transaction Analysis Dashboard

## Project Overview

This project presents an interactive retail transaction dashboard developed using Google Looker Studio. The dashboard helps stakeholders monitor business performance, identify sales trends, evaluate customer behavior, and support data-driven decision making.

## Business Requirements

In this case study, a retail company recently implemented a transaction recording system. The Product Manager requested an interactive dashboard to:

1. Monitor overall transaction performance.
2. Analyze sales trends over time.
3. Evaluate payment method performance.
4. Identify top-selling products.
5. Analyze discount spending across categories.
6. Review transaction summaries by product and customer.

## Key Performance Indicators (KPIs)

| Metric | Description |
|---------|-------------|
| Transactions | Total completed transactions |
| Revenue | Total sales revenue |
| Quantity | Total products sold |
| Discount | Total discounts provided |
| Profit | Total profit generated |

## Key Insights

### Sales Performance
Revenue increased significantly throughout 2022 before declining toward the end of the period, indicating potential seasonal effects or changes in customer demand.

### Payment Method Analysis
COD contributed the highest transaction volume and revenue, indicating strong customer preference for direct payment methods.

### Product Performance
IDROID_BALRX7_Gold was the best-selling product and significantly outperformed other products.

### Discount Allocation
The Entertainment category received the highest discount allocation among all categories.
